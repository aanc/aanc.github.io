How to solve Git error HTTP 413

nginx git

Oct 10, 2013

If you have the following error when you push your repository to a remote : error: RPC failed; result=22, HTTP code = 413 fatal: The remote end hung up unexpectedly fatal: The remote end hung up unexpectedly The issue may not be linked to git directly, but to your webserver, which may not be able to handle big files upload. Adding the client_max_body_size 50m; directive to the http section of your nginx.

How to use nginx as a reverse proxy

nginx

Oct 10, 2013

Open the configuration file of your nginx’s site, usually located in /etc/nginx/sites-available/, and put the following directives in your server block: server { listen 80; server_name your-domain.com; location / { proxy_pass http://localhost:2368; } } This example will redirect all HTTP requests recieved on your-domain:80 to local port 2368.
